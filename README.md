# LyricsPlayer
> 기타 음악 스트리밍 서비스의 실시간 가사 스타일의 영감을 받은 가사 플레이어입니다.

## 개요
LyricsPlayer는 `C:\Users\USER\Videos` 폴더에 있는 MP4 파일을 자동으로 읽고,  
같은 이름의 `.lrc` 또는 `.srt` 가사 파일을 불러와 재생 시간에 맞춰 가사를 표시하는 플레이어입니다.

## 지원 파일 형식
- MP4
- LRC 또는 SRT

## 동작 방식
1. 지정된 폴더에서 MP4 파일 검색
2. 동일한 파일명의 LRC 또는 SRT 자동 로드
3. 재생 시간 기준으로 가사 표시
